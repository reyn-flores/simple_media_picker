# Changelog

All notable changes to this project will be documented in this file.

## 0.0.1

### Added
* Initial release
* Album browsing with thumbnail previews
* Multi-selection support for photos and videos
* Lazy loading with pagination
* Customizable confirm button text
* Material 3 design

## 1.0.0

### Added
* Major feature: toggle to enable or disable multiple selection

## 1.0.1

### Fixed
* Optimize pagination to prevent concurrent requests and improve scroll and thumbnail performance
